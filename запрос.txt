Посмотри на текущий код приложения во всех папках. Приложение является прототипом игры для AR в браузере. Оно сделано с помощью vite, командой npm run build я запускаю билд приложения. В папке dist содержимое билда. Далее я загружаю его в репозиторий git (всё содержимое папки dist). Я это делаю чтобы создавать GitHub Pages страницу на которой я могу тестировать приложение. Как можно было бы автоматизировать процесс, сделать репозиторий который себе в корень мог бы сохранять содержимое папки dist.
